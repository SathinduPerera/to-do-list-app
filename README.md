<h1>A To-Do List</h1>

<p>A to do list app that allows the user to.</p>

<ul>
  <li>Create tasks. Add a title, add a deadline, and a small description.</li>
  <li>Update set tasks details</li>
  <li>Delete tasks</li>
</ul>

  <li>Node.JS</li>

  <li>SQLlite</li>
</ol>
